Descrizione

Questo sistema gestisce e analizza i dati provenienti da braccialetti Empatica E4 (dataset FatigueSet), con pipeline simulata in tempo reale.

Come leggiamo i CSV
  1. Il client o il feeder apre i file CSV: wrist_acc.csv, wrist_bvp.csv, wrist_eda.csv, wrist_hr.csv, wrist_ibi.csv, wrist_skin_temperature.csv.
  2. Leggiamo una riga per volta (saltando l’header). Ogni riga deve avere timestamp,value oppure solo value.
  3. Se il timestamp non è presente, usiamo l’ora corrente (time.time()).
  4. Invio via HTTP POST a /api/sensor_data con payload JSON: {username, sensor, timestamp, value}.
  5. Il server salva su SQLite e calcola la media mobile per rilevare anomalie sopra soglia.
